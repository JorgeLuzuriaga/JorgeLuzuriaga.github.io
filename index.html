Bienvenido al Proyecto Django

Este proyecto es un ejemplo básico de cómo utilizar Django con una plantilla y cómo subirlo a GitHub.

Características del Proyecto

  • Implementación básica de Django.
  • Uso de plantillas de Django.
  • Configuración para ser desplegado en GitHub Pages.

Cómo Configurar el Proyecto

Sigue estos pasos para configurar y ejecutar el proyecto localmente:


# Clona el repositorio
git clone https://github.com/tu-usuario/django-template-project.git

# Navega al directorio del proyecto
cd django-template-project

# Crea y activa un entorno virtual
python -m venv env
source env/bin/activate  # En Windows usa: env\Scripts\activate

# Instala las dependencias
pip install -r requirements.txt

# Realiza las migraciones
python manage.py migrate

# Ejecuta el servidor de desarrollo
python manage.py runserver
